公開から何日目？の実装
------------------------------------------------------------------------------------------------

.. code-block:: python

    >>> from datetime import date
    >>> AINOUTA_DAY = date(2021, 10, 29)
    >>> date.today() - AINOUTA_DAY
    datetime.timedelta(days=22)
    >>> (date.today() - AINOUTA_DAY).days + 1  # 10/29を1日目とする
    23
